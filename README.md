# Registro-de-clientes-de-un-banco-en-C++
## 📌 Descripción
En este programa se aborda el uso de conceptos fundamentales de la programación orientada a objetos, específicamente clases y herencia, aplicados a un sistema de registro para un banco.

## 🛠 Tecnologías utilizadas
C++

## ⚙️ Funcionalidades
- Registro de clientes
- Registro de cuentas
- Modificacion de informacion
- Consulta

## ▶️ Cómo ejecutar el proyecto
1. Clonar el repositorio
2. Importar el archivo a un editor de codigo
3. Ejecutar en servidor local
